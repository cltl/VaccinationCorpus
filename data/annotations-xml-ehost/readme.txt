This folder contains the annotations in xml format generated by the annotation tool e-Host (https://github.com/chrisleng/ehost)

Attribution - 294 files annotated by the main annotator, 25 files annotated by a secondary annotator to calculate IAA.

Claims - 294 files annotated by the main annotator, 50 files annotated by a secondary annotator to calculate IAA.

Events - 294 files annotated by the main annotator, 5 files annotated by a secondary annotator to calculate IAA.
